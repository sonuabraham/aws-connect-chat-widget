AWS Connect Chat Widget Integration Example

1. Basic Integration

Add this script tag to your HTML page:

<script src="https://cdn.example.com/integration.js"></script>

2. Custom Configuration

Initialize with your AWS Connect settings:

<script> // Configure before the widget loads window.AWSConnectChatWidgetAPI.init({ aws: { region: 'us-east-1', instanceId: 'your-instance-id', contactFlowId: 'your-contact-flow-id', apiGatewayEndpoint: 'your-api-gateway-endpoint' }, ui: { theme: { primaryColor: '#007bff', secondaryColor: '#6c757d' }, position: { bottom: '20px', right: '20px' }, messages: { welcomeMessage: 'Hello! How can we help you today?' } } }); </script>

3. Manual Control

Control the widget programmatically:

// Update configuration AWSConnectChatWidgetAPI.updateConfig({ ui: { theme: { primaryColor: '#28a745' } } }); // Get widget state const state = AWSConnectChatWidgetAPI.getState(); // Destroy widget AWSConnectChatWidgetAPI.destroy();
